# Installation

## System Requirements

- **Operating System**: Windows, macOS, or Linux
- **Git**: Must be installed and available in system PATH (for repository operations)
- **Internet Connection**: Required for API integrations
- **Disk Space**: ~500MB for application and data storage

## Download and Install

1. **Download the Installer**
   - Download `Axis Setup.exe` from the [latest release](https://github.com/zkyko/fh-Axis/releases)
   - The installer file is typically 80-90MB

2. **Run the Installer**
   - Double-click `Axis Setup.exe`
   - Follow the installation wizard
   - Choose installation directory (default is recommended)
   - Complete the installation

3. **Launch Axis**
   - Launch Axis from the Start Menu or Desktop shortcut
   - The application window should open with the Dashboard displayed

## First Launch

On first launch, you should see:

- ✅ Sidebar navigation on the left
- ✅ Dashboard in the main content area
- ✅ No errors or crash dialogs

The application is ready for configuration. Next, proceed to [Configuration](./configuration.md) to set up your integrations.
